Homework 0

DUE DATE: Sunday Jan 28 at 11:59pm

Be sure to perform all of the tasks.

Task 1: Join Gradescope

You should have received an invitation to join. Homework and project submissions will be done through this website.

Task 2: Configure your computer

Install and configure vscode along with extensions

Download and install vscode along with the following extensions

  • Python by Microsoft
  • Pylint by Microsoft
  • Live Share Extension Pack by Microsoft

Run a terminal

Get familiar with using the terminal to create, list, copy, remove files and navigating the directories.

  1. Explore the directories on your computer using the terminal. Relevant commands are cd and pwd
  2. Create a file called one.py with vscode and save it. Put the file in your home directory
  3. List contents of a directory using the command ls
  4. Create a directory called hw0 in your home directory using the command mkdir hw0
  5. Navigate to the new directory using the command cd hw0
  6. Move one.py to hw0 using the command mv ..\one.py .
  7. Copy one.py and call the new file two.py using the command mv one.py two.py
  8. List the contents of the current directory
  9. Figure out what directory you currently are in.
  10. Remove one.py using the command rm one.py
  11. List the contents of the current directory again

Install python3

Download and install python3.

  • Test your installation by running it in the terminal using the command python3
  • Exit out of the python interpreter by typing at the python prompt the command exit()

Task 3: Submit hello.py

  • Create a file called hello.py. This is your first python script. It simply prints two lines. The first line is your first name followed by your last name. The second line is your student ID.
  • Save the file and test it by running python3 hello.py
  • Submit the file to gradescope as your hw0
