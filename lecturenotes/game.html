  1. Example: Create a Player class that has the following attributes: name, level, experience, health, and attack power. Create methods to increase the player’s experience and health, and to attack another player.

Practice Questions:

  • How would you create an instance of the Player class?
  • How would you access the health attribute of a Player instance?
  • How would you call the attack method of a Player instance?
  1. Example: Create a Weapon class that has the following attributes: name, damage, and durability. Create methods to use the weapon to attack another player and to decrease the weapon’s durability.

Practice Questions:

  • How would you create an instance of the Weapon class?
  • How would you access the damage attribute of a Weapon instance?
  • How would you call the attack method of a Weapon instance?
  1. Example: Create a Game class that has the following attributes: name, players, and weapons. Create methods to add players and weapons to the game, to start the game, and to end the game.

Practice Questions:

  • How would you create an instance of the Game class?
  • How would you add a player to a Game instance?
  • How would you start a Game instance?
  1. Example: Inheritance Create a class called Enemy that inherits from the Player class. The Enemy class should have the same attributes as the Player class, but with a different attack method.

Practice Questions:

  • How would you create an instance of the Enemy class?
  • How would you access the health attribute of an Enemy instance?
  • How would you call the attack method of an Enemy instance?
  1. Example: Polymorphism Create a class called Boss that inherits from the Enemy class. The Boss class should have a different attack method than the Enemy class. Create a function that takes a Player, Enemy, or Boss instance as an argument and calls their attack method.

Practice Questions:

  • How would you create an instance of the Boss class?
  • How would you call the function with a Player instance as an argument?
  • How would you call the function with a Boss instance as an argument?
